# 📊 Telecom Customer Churn Analysis

## 📌 Project Overview

This project focuses on analyzing telecom customer data to identify key factors contributing to customer churn. The goal is to derive actionable insights that help businesses improve customer retention and reduce revenue loss.

---

## 🛠️ Tools & Technologies

* Python (Pandas, Matplotlib)
* SQL (MySQL)
* Power BI

---

## 📂 Dataset

* Contains ~7,000 customer records
* Features include:

  * Customer demographics
  * Contract type
  * Tenure
  * Monthly charges
  * Payment method

---

## 🔍 Project Workflow

### 1️⃣ Data Cleaning (SQL)

* Handled missing values
* Corrected data types
* Removed inconsistencies

### 2️⃣ Exploratory Data Analysis (Python)

* Analyzed churn distribution
* Studied impact of:

  * Contract type
  * Tenure
  * Payment method
  * Monthly charges

### 3️⃣ Dashboard Development (Power BI)

* Built an interactive dashboard with:

  * Churn Rate
  * Total Customers
  * Revenue Loss
* Added slicers for dynamic filtering

---

## 📈 Key Insights

* Customers with **month-to-month contracts** have higher churn
* Customers with **low tenure** are more likely to leave
* Certain **payment methods** show higher churn behavior

---

## 💡 Business Recommendations

* Focus on retaining high-risk customers
* Encourage long-term contracts
* Improve engagement for new customers
